# E-commerce-Global-Performance-Matrix-Revenue-User-Trust-Analysis
Цей аналітичний звіт був розроблений для оцінки комерційної ефективності платформи в розрізі континентів. Проєкт поєднує фінансові показники (виторг) із технічними даними (типи пристроїв) та метриками якості аудиторії (верифікація акаунтів).
